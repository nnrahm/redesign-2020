About Joomla!

Joomla! is a free and open-source content management system (CMS) initially released in September 2005. Developed by Open Source Matters Inc., it is built on a model-view-controller web application framework that can be used independently of the CMS and allows you to build powerful online applications.

I found Joomla to be a great tool that was used as the back end engine of our customer facing website. As someone with some technical skills, I was able to learn how to use the editing tools pretty quickly, I learned how to be a user administrator and set up permissions for our customer portal to allow ease of access for customer materials - the overall set up of these features was a very stress free process.

Installing Joomla!

Below are instructions on how to get started with Joomla!. You may also find more detailed information about installation and configuration here. Before successfully installing Joomla!, there are several prerequisites needed. Technical requirements can be found here.

Preparing for Installation

Download and Upload

Download the current release of Joomla!. Use a FTP (File Transfer Protocol) Client to move the Joomla download package to your server and unpack it to install the files. Make sure you are using a FTP Client's official release to avoid running into errors or other problems.

Database Requirements

If you need to create a database, you can read through this article first, or skip to the next step. You will need to note basic database information needed when the actual Joomla! installation has been initiated:

  • Location of database, localhost, or a specific host's server such as dbserver1.yourhost.com
  • Database name
  • Database user's name
  • Database user's password

Starting the Installation

Configuration

When the above requirements have been met, you are officially ready to install Joomla!. You can start the web installer by opening your browser and going to your site's domain name. On host installation, you will use http://www.yoursitename.com. If installing locally, use http://localhost/<path to Joomla files>, and you should see the installation screen.

Fill in the Blanks

Your site name can be changed at any point in the Site Global Configuration package. The description is a global fallback meta description used on every page, which will be used by search engines. Generally a maximum of 20-25 words is optimal, but this can also be changed in the Site Global Configuration page at any time. Enter a valid email address in case you forget your password. This is the email that will be used to reset or change the admin password. Make sure your password is strong and hard to guess. Administrators have maximum front and backend control of the site. Site Offline gives you the option to display a 'Site is offline' message when you browse to yoursitename.com to view the homepage. If you select no, site will be live when you view the homepage.

Database Configuration

Configuration Settings

During this section, you will need to enter information about the database you will use for Joomla!:

  • Database Type: MySQLi is the common database used
  • Hostname: Where is your database located? Common is localhost, but some hosts use a specific database server such as dbserver1.yourhost.com
  • Username: The username used to connect to the database
  • Password: The password for the database's username
  • Database Name: the name of the database
  • Table Prefix One is generated automatically, but you can change it. For example, jos3_ can be used. Just don't forget to put the underscore character (_) at the end of the prefix.
  • Old Database Process: Should the installer backup or delete existing tables during the installation of new tables? Click, Yes or No to select the choice.

These choices can be edited on the Site Global Configuration page after the installation has been completed. Making changes to these setting after installation will break the site unless you have a complete copy of the current database being used by the Joomla! installation.

Sample Data

Install Sample Data

If you are new to Joomla!, it will be beneficial to install sample data to see how Joomla! works. You can choose to have the configuration settings emailed to you. If the Email Configuration choice is selected, the Email Password choice will appear. The email password is off by default for security. You can choose to have the password included, click Yes. Afterwards, it will be time to check the configurations of your install and the environment of the installation.

Check the Configuration

If everything is in order, you will see the installation at the top of the overview page. If not, this is the time to check what might be causing the issue.

Main Configuration

All the website specific information, such as the website name, description, admin username, etc.

Database Configuration

Your database configuration contains the information about the database Joomla! will use.

Pre-Installation Check

These requirements must all be shown as Yes, otherwise you will not be able to install Joomla! With the exception of the PHP Version, the rest are usually controlled in the php.ini. You may need assistance from your host in correcting these settings or checking to see if it is possible to adjust them.

PHP Settings

Typical PHP settings might cause the install to fail and may need adjusting include the following with suggested values: (1) memory_limit (64M), (2)max_execution_time (300), (3) post_max_size (30M), (4) upload_max_filesize (30M). For more information, see the PHP configuration file.

Recommended Settings

These are settings are recommended in your PHP configuration, but will not prevent Joomla! from being installed. You can refer to the above instructions on how they may be changed.

Finalizing the Installation

Congratulations! Joomla! should now be properly installed. If you'd like to start using Joomla! without installing any extra languages, there is one last step to really complete the installation: you must delete the Installation Folder. To do this, click on Remove Installation folder and a success message will appear. You can then navigate to the Administrator log in or go right to your site by clicking Site.

Installing Languages

To install extra languages, before deleting the Installation folder, click on Extra steps: Install languages. This will continue the installation of Joomla! by taking you to a new installation page. To choose a default language, when the download is complete, you can choose the default for the Site and Administrator interface. You may also choose to activate the multilingual features of Joomla! at this time too.

Finalize

You should now be presented with a congratulations page showing Joomla! has now been installed. The difference will be a notation of the Administrator and Site language settings if they were set. You will also not see the Extra Steps message in the bottom right. You can also delete the Installation Folder now.
